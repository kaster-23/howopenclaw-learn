OpenClaw has a complete voice stack: speech-to-text for input, an AI model for processing, text-to-speech for output, and an optional voice wake word for hands-free activation. Each component is independent — you can mix and match based on your needs and budget.

## The Voice Stack

### Talk Mode

The foundation. Talk Mode creates a hands-free conversation loop:

1. Microphone captures your speech
2. Whisper transcribes it to text
3. Your agent processes it
4. ElevenLabs speaks the response aloud

Activate via the web client, a channel command, or a voice wake word.

### Text-to-Speech (ElevenLabs)

> "ElevenLabs produces remarkably natural-sounding speech, far beyond the robotic voices you might remember from older TTS systems."

You choose a voice from ElevenLabs' library or clone your own. Configure it in your agent's SOUL.md to match personality.

Costs approximately $5/month on their starter plan for typical usage.

### Speech-to-Text (Whisper)

OpenAI's Whisper converts your audio to text. It supports 50+ languages at approximately $0.006 per minute — roughly $0.36 per hour of conversation.

Audio is processed locally until a voice wake is detected, then sent to Whisper for transcription.

### Voice Wake

Hands-free activation using a custom wake word. The wake word detection runs **locally on your device** — audio only reaches cloud services after the wake word is detected.

Configure in `openclaw.json`:

```json
{
  "voice": {
    "wake": {
      "enabled": true,
      "wakeWord": "hey assistant",
      "sensitivity": 0.5,
      "modelPath": "~/.openclaw/models/wake-word/"
    }
  }
}
```

Sensitivity ranges from 0.0 (few false positives, may miss genuine wake words) to 1.0 (catches everything, more false positives). Start at 0.5.

### Twilio (Phone Integration)

Connect your agent to a real phone number. Inbound calls get answered by your agent; it can also make outbound calls.

Costs approximately $0.013/minute for outbound US calls plus a Twilio number (~$1/month).

```json
{
  "voice": {
    "twilio": {
      "enabled": true,
      "accountSid": "${TWILIO_ACCOUNT_SID}",
      "authToken": "${TWILIO_AUTH_TOKEN}",
      "phoneNumber": "+15551234567"
    }
  }
}
```

## Setup Path

Most users follow this progression:

**Step 1: Talk Mode + Whisper STT**
No ongoing cost beyond Whisper's per-minute rate. Test that the voice workflow feels right before adding TTS.

**Step 2: Add ElevenLabs TTS**
~$5/month. This is the point where Talk Mode starts feeling genuinely natural.

**Step 3: Enable Voice Wake**
Hands-free activation. Now you can talk to your agent without touching a device.

**Step 4 (Optional): Add Twilio**
Only if you need actual phone call integration.

## Talk Mode Configuration

```json
{
  "voice": {
    "talkMode": {
      "enabled": true,
      "stt": "whisper",
      "tts": "elevenlabs",
      "language": "en",
      "continuous": false,
      "silenceDetection": {
        "threshold": 1500,
        "silenceMs": 800
      }
    },
    "elevenlabs": {
      "apiKey": "${ELEVENLABS_API_KEY}",
      "voiceId": "your-voice-id",
      "model": "eleven_monolingual_v1"
    },
    "whisper": {
      "apiKey": "${OPENAI_API_KEY}",
      "model": "whisper-1",
      "language": "en"
    }
  }
}
```

`silenceMs: 800` means the agent waits 800ms of silence before treating your speech as complete. Increase this if it cuts you off; decrease if it waits too long.

## Voice-Specific SOUL.md Instructions

Add voice-specific behavior to your SOUL.md:

```markdown
## Voice Behavior
When responding via voice:
- Keep responses to 1-3 sentences unless asked for more
- No markdown, bullet points, or code blocks (they don't translate to speech)
- For longer answers, offer to send a written summary to iMessage
- When someone leaves a voice note, transcribe it and respond in text
```

## Handling Voice Notes

With Whisper installed, your agent can transcribe voice messages sent via WhatsApp or iMessage:

```markdown
## In SOUL.md
When you receive a voice note:
1. Transcribe it
2. Respond to the content
3. Include the transcription at the bottom of your response so I can reference it
```

## Processing Voice Brainstorms

A useful pattern: speak through a messy idea, agent structures it.

```markdown
## In SOUL.md
When I say "organize this" after a brainstorm:
- Extract the key ideas
- Group them into themes
- Identify action items
- Send a structured summary to Slack
```
