## Architecture

```
You → Channels → Gateway (ws://127.0.0.1:18789) → Agent → Skills/Memory/APIs
```

Files live in:
- `~/.openclaw/` — config and identity files
- `~/.openclaw/workspace/` — workspace (documents, downloads, agent-accessible files)

---

## Configuration Files

| File | Purpose |
|---|---|
| `~/.openclaw/SOUL.md` | Agent personality, style, values |
| `~/.openclaw/USER.md` | Info about you |
| `~/.openclaw/IDENTITY.md` | Agent name and appearance |
| `~/.openclaw/MEMORY.md` | Persistent knowledge base |
| `~/.openclaw/MESSAGING.md` | Channel routing rules |
| `~/.openclaw/HEARTBEAT.md` | Proactive idle behaviors |
| `~/.openclaw/openclaw.json` | System settings and keys |

## CLI Commands

### Gateway Management

```bash
openclaw gateway                       # Start gateway (foreground)
openclaw gateway install && openclaw gateway start  # Install and start as background service
openclaw gateway stop                  # Stop gateway
openclaw gateway restart               # Restart
openclaw gateway status                # Check if running
openclaw logs                          # View recent logs
openclaw logs --follow                 # Stream live logs
openclaw doctor                        # Run diagnostics
```

### Skills

```bash
openclaw skills install @author/name     # Install a skill
openclaw skills install @author/name --sandbox  # Install with isolation
openclaw skills list                     # List installed skills
openclaw skills info @author/name        # Skill details
openclaw skills update --all             # Update all skills
openclaw skills remove @author/name      # Uninstall
```

### Channels

```bash
openclaw channels add imessage           # Configure a channel
openclaw channels test imessage          # Test connection
openclaw channels list                   # List configured channels
openclaw channels disable imessage       # Disable without removing
```

### Memory

```bash
openclaw memory list                    # List memory entries
openclaw memory search "query"          # Search memory
openclaw memory add "key: value"        # Add entry
openclaw memory remove "key"            # Remove entry
openclaw memory compact                 # Compress old entries
```

### System

```bash
openclaw gateway install                # Install as system service
openclaw gateway start                  # Start the system service
openclaw gateway stop                   # Stop the system service
openclaw gateway restart                # Restart the system service
openclaw secrets set KEY value          # Store encrypted secret
openclaw secrets list                   # List secret names
openclaw costs                          # View API cost breakdown
openclaw onboard --reconfigure          # Re-run setup wizard
```

## Channel Quick Configs

```json

  "channel": "imessage",
  "enabled": true,
  "dmPolicy": "allowlist",
  "allowedContacts": [
    "+15551234567"
```

## Cron Syntax

```
┌─ minute (0-59)
│ ┌─ hour (0-23)
│ │ ┌─ day of month (1-31)
│ │ │ ┌─ month (1-12)
│ │ │ │ ┌─ day of week (0-7, 0=Sun)
* * * * *
```

| Expression | Meaning |
|---|---|
| `0 7 * * 1-5` | 7 AM weekdays |
| `0 9 * * *` | 9 AM every day |
| `0 17 * * 5` | 5 PM every Friday |
| `*/30 * * * *` | Every 30 minutes |
| `0 0 1 * *` | Midnight on the 1st of each month |
| `0 8,12,17 * * 1-5` | 8 AM, noon, 5 PM on weekdays |

## Troubleshooting

| Problem | Check |
|---|---|
| Gateway won't start | `openclaw logs` for error details |
| Agent not responding | `openclaw gateway status` → gateway running? |
| Channel not working | `openclaw channels test [name]` |
| API errors | Check key validity, quota limits |
| Memory not persisting | Verify MEMORY.md path in config |
| Cron not firing | Check timezone in `openclaw.json` |
| High costs | `openclaw costs --breakdown` |
| Slow responses | Check model tier, context length |
